python2とpython3を共存させる

Twitter Facebook Hatena

モチベーション

  • pythonとpython3とpipとpip3をいちいち使い分けるのが面倒くさい．
  • むやみやたらにpipすると環境がどんどん汚れていくのが嫌だ．
  • プロジェクトごとにDockerコンテナを立ち上げるような，大げさな管理はしたくない．

venvでsandboxをつくる

$ python -V
Python 2.7.10
$ python3 -V
Python 3.7.3
$ mkdir ml
$ cd ml
$ python3 -m venv venv
$ . venv/bin/activate
(venv) $ python -V
Python 3.7.3

venv下にあるパッケージを確認する

(venv) $ pip freeze

venvを解除する

(venv) $ deactivate
$ python -V
Python 2.7.10
